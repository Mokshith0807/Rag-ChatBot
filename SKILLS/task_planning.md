# Task Planning Agentic Skill

## Description
Ability to break complex problems into structured steps and execute them efficiently.

## Capabilities
- Decompose tasks into smaller actionable steps
- Prioritize work based on importance
- Create clear execution plans
- Track progress systematically

## Example Use Case
Building a RAG chatbot by dividing into:
1. Data collection  
2. Document chunking  
3. Embedding generation  
4. Vector database setup  
5. Retrieval and response generation
